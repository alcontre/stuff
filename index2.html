▼ Block: File Header
▼ Block: Magic Number
▼ Block: Signature Bytes

Bit LayoutDescription
"BINF"
ASCII encoding of magic bytes (0x42 0x49 0x4E 0x46)
▼ Block: Version
▼ Block: Major Version
▼ Block: Version Number
Bit LayoutDescription
[Version: 1B]
Major version number (e.g., 0x01 for version 1)
▼ Block: Minor Version
Bit LayoutDescription
[Revision: 1B]
Minor version/revision number
▼ Block: Metadata
Bit LayoutDescription
[File Size: 4 bytes]
Total file size in bytes
[Checksum: 4 bytes]
CRC32 checksum of file content
▼ Block: Data Section
▼ Block: Metadata Block
▼ Block: Block Header
▼ Block: Type Info
Bit LayoutDescription
[Type: 1B]
Block type identifier (1=metadata, 2=content, 3=index)
[Flags: 2b]
Control flags (bit 0=compressed, bit 1=encrypted)
▼ Block: Block Length
Bit LayoutDescription
[Length: 3B]
Block length in bytes (24-bit, max 16MB)
▼ Block: Timestamp
Bit LayoutDescription
[Timestamp: 8B]
Unix timestamp (64-bit seconds since epoch)
▼ Block: Content Block
Bit LayoutDescription
[Record Header: 2B]
Record type and flags (2 bytes)
[Record Data: Variable]
Variable-length record payload
[Compression: 1b]
Compression flag (0=uncompressed, 1=compressed)
▼ Block: Footer
▼ Block: Index Table
Bit LayoutDescription
[Entry Count: 2B]
Number of index entries
[Entry Size: 1B]
Size of each entry in bytes
[Entries: 8B each]
Index entries (offset and length pairs)
▼ Block: End Marker
Bit LayoutDescription
[Terminator: 4B]
End-of-file marker (0xFFFFFFFF)
[Checksum: 4B]
Final verification checksum
